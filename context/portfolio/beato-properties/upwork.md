# Real Estate Rebrand & Website — Beato Properties

## Overview
Beato Properties came to me for a full rebrand and website relaunch. Starting from a blank slate, I delivered a complete brand identity and a multilingual WordPress site built to list, filter, and sell properties — with direct booking integration and a custom data migration from their previous site.

## The Challenge
The client's existing site (Cirh.es) no longer reflected the business they wanted to be. They needed a new name, a new identity, and a new digital home — all built to support a Spanish-speaking audience across Catalan and Castilian, with online booking for property viewings baked in from day one.

## What I Did
- Created a full brand identity: custom logo, colour palette, brand voice, and a brand guide as the canonical reference
- Designed static mockups for the home page and property listing page before a single line of code was written
- Built a WordPress site using a custom Divi child theme with Estatik for property management
- Configured WPML for multilingual delivery in Spanish and Catalan
- Integrated FluentBooking and FluentForms so agent profile cards link directly to the right booking form
- Built a custom sold-property carousel component
- Bulk-migrated all property listings from the old site
- Wired up Google Maps, Gmail, and Calendar APIs via Google Cloud Console
- Delivered full site documentation and a 1:1 training session with the client

## Results
Full-stack rebrand and launch delivered — from blank brief to live multilingual site. Client walked away with a brand guide, a trained team, and a documented site they can manage independently.

## Skills & Tools
WordPress, Divi, Estatik, FluentBooking, FluentForms, WPML, Google Maps API, Gmail API, Google Calendar API, Brand Identity, Logo Design, UI Mockups
