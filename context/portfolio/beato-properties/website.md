# Beato Properties — Brand Identity & Website

**Client:** Jaume Beato  
**Year:** 2026  
**Services:** Brand Identity, UI Design, WordPress Development, Systems Integration

---

## The Brief

Beato Properties was starting fresh. The client was relaunching their real estate agency under a new name — stepping away from the existing Cirh.es brand and building something with more presence and purpose. They needed a complete brand identity, a website that could handle a live property catalogue, and a way for prospective buyers to book viewings directly through the site. The audience spans two languages: Spanish and Catalan.

## Approach

We started with discovery — an intake interview and brand questionnaire to understand Jaume's vision, the properties he works with, and the clients he wants to attract. From that, the brand took shape: a custom logo, a considered colour palette, and a brand voice guide that would anchor every piece of copy going forward. The brand guide became the single source of truth before a single page was designed.

Mockups followed for the home page and a representative property listing page — static designs to align on layout and feel before committing to build. With those signed off, we moved into development.

The site runs on WordPress with a custom Divi child theme and Estatik for property management. All existing listings were bulk-migrated from the old site. Agent profile cards connect directly to dedicated booking forms via FluentBooking and FluentForms, so a viewer can go from browsing a property to scheduling a visit in two clicks. Google Maps, Gmail, and Calendar APIs are wired in via Google Cloud Console — infrastructure the client set up themselves with our guidance.

WPML handles the multilingual layer, with full Spanish and Catalan versions of the site. We closed the project with complete site documentation and a live 1:1 training session with Jaume.

## Key Deliverables

- Custom logo, colour palette, and brand voice
- Brand guide
- Home page and property listing page mockups
- WordPress site — custom Divi child theme
- Estatik property management setup
- Bulk property data migration
- Custom sold-property carousel
- FluentBooking + FluentForms agent booking integration
- Google Maps, Gmail, and Calendar API integration
- WPML multilingual site (Spanish + Catalan)
- Site documentation and training session

## The Result

A new brand and a fully operational property website, delivered from first brief to live launch. Beato Properties now has a digital presence that matches their ambitions — a site they can manage, a brand they can grow, and a booking system that works from day one.
