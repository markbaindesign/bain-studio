When Aura is built, her onboarding checklist should include setting up a Google Cloud API key as a prerequisite. Both APIs are free and live in the same project:

1. PageSpeed Insights API — needed for CWV/performance scores in seo-audit. Without it, PSI calls 429.
2. Google Search Console API — needed for keyword rankings, impressions, CTR per page, URL indexing status. GSC is read-only and free; no equivalent alternative exists.

One API key covers both. Setup: console.cloud.google.com → enable both APIs → Credentials → Create API Key → store in project .env.

BSTD-036 tracks getting this key for NORE. Aura's SKILL.md should check for the key on first run and warn clearly if it's missing, rather than failing silently or hitting rate limits.
