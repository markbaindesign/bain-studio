Full context for building Aura (studio SEO god). Developed during live SEO audit on NORE (thenatureofrealestate.com) on 2026-06-17/18. Use as design brief.

## What Aura is
A studio skill for 'how do we improve rankings' — not 'does this page pass a technical check' (seo-audit's job). Interprets audit data, sets strategy, drafts copy, produces prioritised action plans. Called per-project.

Model: Sonnet. Name: Aura (goddess of the breeze).

## Free tool stack
1. Ahrefs Webmaster Tools — site audit, CSV exports, monthly cadence
2. Google Search Console — rankings, impressions, CTR, index coverage, CWV field data; API available
3. seo-audit skill / seo_audit.py — robots.txt, sitemap, per-page HTML metadata, PSI scores

## Onboarding requirement
Check for Google Cloud API key on first run. One key covers PSI + GSC APIs. Fail loudly if missing — not silently.

## Core strategy
GSC impressions × Ahrefs CSV = prioritised fix list.
- High impressions + low CTR → bad title/meta → fix first
- High impressions + low position (8–15) → close to page 1 → one edit could move it
- Zero impressions + indexed → orphan/thin → cull or consolidate

Fix priority order: (1) redirect chains, (2) 4XX with external backlinks, (3) 4XX with internal inlinks, (4) orphan pages, (5) missing meta on high-impression pages, (6) oversized images, (7) multiple H1s.
